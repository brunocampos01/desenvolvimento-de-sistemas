# How to README?

It must have the following format:
- Title 
- Tags: Search shields [here !](https://shields.io/#/examples/build)
- Describe project 
- Pre Requirements
- Quickstart
- Install
- Examples
- References 
- Bug fix or sugestions
- How to contribuite </br> 
- Author
- Copyright and license
